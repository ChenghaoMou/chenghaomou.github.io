Supported types of data storage:
- Redshift (column based)
- RDS or Aurora (row based)
- DynamoDB (NoSQL)
- [[S3]]
- ElasticSearch / [OpenSearch](https://aws.amazon.com/opensearch-service/)
- [Amazon ElastiCache](https://aws.amazon.com/elasticache/)
