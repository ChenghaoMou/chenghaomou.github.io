Rank-One Model Editing

As described in ^9eca9d, we now can apply changes to model weights to take in a new “fact”.

By treating a linear projection as [Associative Memory], we can update the weights to take in a new fact by calculating the following:

W^k∗​=v∗​

Where W^ is the new weight matrix, k∗​ is the query vector for the subject, and v∗​ is the value vector for the new fact (under the same subject and relation).

k∗​ can be calculated by averaging hidden states over different context for a given subject beforehand, which can be seen as a universal subject query vector.

v∗​, on the other hand, can be calculated by finding a z that minimizes the loss:

z is defined as the output hidden state at layer l and step i. (This seems like some computation)

After we have both k∗​ and v∗​, we can calculate the updated W^ and plug it back into the model.
