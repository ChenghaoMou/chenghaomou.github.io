Speculative Decoding

Source: Fast Inference from Transformers via Speculative Decoding

Decoding easy tokens should not require a full forward pass of a dense model, as this can be achieved using a relatively small one at a lower cost. Consider a large model, denoted as p, and a small model, denoted as q.

The process is as follows:

  1. Generate γ tokens using q;
  2. Evaluate those tokens using p with token probabilities and distributions. Additionally, an new token distribution after γ tokens is also available;
  3. Accept the prediction xt​ of q if pq​(xt​)≤pp​(xt​) or accept with a probability pq​(xt​)pp​(xt​)​.
  4. If rejected, then sample xt​ again from the (Pp​(x)−Pq​(x))+​. This is proven to be equivalent to sampling directly from p;

This process is guaranteed to generate at least one token from p and at most γ+1 tokens, all in one go.

When adopting this method, there are a few parameters that require attention:

  1. How good is the small model at approximating the large one’s distribution α?
  2. How many tokens are outsourced to the small model γ?
  3. How cost-effective is the small model compared to the large model c?
