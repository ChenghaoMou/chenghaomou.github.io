# PyTorch Ecosystem Day 2021 Highlights

- Platforms, Ops, and Tools
	- `torch_ort` Using one line of code to adopt ONNX runtime for your model training
	- `dask` extends their distribution capability into PyTorch training, so does `ray`
	- `ClearML`: An open-source solution covering fundamentals of MLOps for R&D — Reproducibility, Orchestration, and Automation
	- `Avalanche`: Continual Learning
	- `TorchDrift`: Drift detection in PyTorch
	- `FairTorch`: Add fairness criteria into your loss

- NLP, Multi-modal
	- `MMF` A modular framework for multi-modal research (specialized encoders for each modality and fusion with transformers/concatenation)
- Performance & Profiler
	- PyTorch Profiler
